# Frontend
## Requirements
- Node 16.17.4 LTS
- NPM 8.x

## Installation
```bash
$ npm install
```

## How to run
### Local
To run the app in development mode locally, run the below command
```bash
$ npm run start
```

The application will spin up on http://localhost:3000/

#### Linting
Apply linting across all the frontend project directories
```bash
$ npm run pretty
```
